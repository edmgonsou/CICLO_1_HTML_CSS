1. Como é possível descobrir qual é a fonte padrão de renderização do navegador?
    RESP.: Basta abrir o DEV TOOL e acessar a opção computer e aparecerá as confiração de fontes do navegador

2. É verdade que quando alteramos um CSS diretametne no DEV TOOL automaticamente alteramos o 
arquivo CSS? 
    RESP.: Não, quando alteramos no DEV TOOL o arquivo CSS não é alterado, se fizer alguma alteração no
        DEV TOOL é somente atualizar a pagina que as configurações gravadas no CSS retornará
